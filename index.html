Spacing System

The toolkit allows you to specify spacing (margin and/or padding) for an element. To specify the spacing direction, use the following class names where $n is the spacing size:

  • os-l-ma-$n all 4 directions
  • os-l-mx-$n left and right
  • os-l-my-$n top and bottom
  • os-l-mt-$n top
  • os-l-mb-$n bottom
  • os-l-ml-$n left
  • os-l-mr-$n right

The spacing system incorporates a scale of 6 steps. Use "0", "nudge", "auto", or the numbers "1" through "6" to specify the amount of space for an element.

Top margin example:

.os-l-mt-0
.os-l-mt-nudge
.os-l-mt-1
.os-l-mt-2
.os-l-mt-3
.os-l-mt-4
.os-l-mt-5
.os-l-mt-6

Auto margin example:

.os-l-mx-auto[style="max-width:400px"]
